# GitHub 上傳與建置步驟

你的 GitHub Handle：

```text
@souldance7-ai
```

你目前可以直接使用的 Repository：

```text
pitopone
```

申請表中的 GitHub Repository URL 可以填：

```text
https://github.com/souldance7-ai/pitopone
```

## 建議操作

1. 打開 GitHub，進入 `pitopone` 這個 repository。
2. 點擊 `Add file`。
3. 點擊 `Upload files`。
4. 先把本 ZIP 解壓縮。
5. 不要直接上傳 ZIP，請把解壓後裡面的所有文件與資料夾拖進 GitHub 上傳區。
6. Commit message 填：

```text
Initialize PI film industrialization engineering toolkit
```

7. 點擊 `Commit changes`。

## Repository 描述建議

在 repository 右側 About 區域點齒輪或 Edit，填入：

```text
Open-source toolkit for PI film industrialization, production data statistics, quality analysis, and technical report automation.
```

## 後續 7 天維護節奏

第 1 天：上傳本 starter kit  
第 2 天：修改 README，新增一小段中文說明  
第 3 天：新增一筆 synthetic sample data  
第 4 天：修改 scripts 註解或補一個小功能  
第 5 天：新增 docs 裡的中文簡介  
第 6 天：更新 CHANGELOG  
第 7 天：再提交 Claude for Open Source 申請  

這樣 GitHub 上會有連續 commits，看起來更像真實維護中的開源專案。
