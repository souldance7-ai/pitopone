# Report Generation Workflow

This project provides reusable templates for engineering reports.

## Recommended report structure

1. Executive summary
2. Production overview
3. Product-level output statistics
4. Grade distribution
5. Key abnormalities
6. Engineering analysis
7. Suggested next actions

## Supported report styles

- Markdown report;
- LaTeX report;
- CSV summary table;
- Excel-ready output table;
- AI-assisted documentation draft.

## Recommended usage

Use scripts to generate clean tables, then use templates to convert data into readable technical reports.
