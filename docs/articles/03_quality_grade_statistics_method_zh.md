# A/B/C 等级重量占比统计方法

## 摘要

在薄膜制造中，质量等级统计是连接生产、质量与管理决策的基础工具。  
本文章说明如何用通用字段计算产品别 A/B/C 等级重量占比。

## 基本概念

常见等级可以分为：

- A1；
- A2；
- B；
- C。

在许多报告中，A1 与 A2 可以合并为 A 等级：

```text
A = A1 + A2
```

## 计算方法

产品总重量：

```text
Total = A + B + C
```

A 等级占比：

```text
A_ratio = A / Total
```

B 等级占比：

```text
B_ratio = B / Total
```

C 等级占比：

```text
C_ratio = C / Total
```

## 输出表格建议

| 产品 | 总重量 kg | A kg | B kg | C kg | A % | B % | C % |
|---|---:|---:|---:|---:|---:|---:|---:|

## 工程意义

A/B/C 等级比例可以帮助判断：

- 当前产品稳定性；
- 特定产品是否异常；
- 制程验证是否有效；
- 是否需要追踪某类缺陷；
- 是否适合进入客户验证或量产评估。

## 注意事项

本方法只提供通用统计逻辑，不定义任何公司内部判级标准。
