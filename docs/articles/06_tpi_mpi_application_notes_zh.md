# TPI / MPI / 复合膜应用场景记录框架

## 摘要

TPI、MPI 与复合膜材料常用于高频高速、柔性电路、封装保护、显示与车用电子等场景。  
本项目不做商业预测，只提供应用场景拆解与工程记录模板。

## 应用场景

| 场景 | 关注点 |
|---|---|
| FPC / FCCL | 尺寸稳定性、剥离强度、介电性能 |
| 高频高速 | Dk / Df、吸湿、可靠性 |
| 封装保护 | 热压残胶、耐热、尺寸变化 |
| 柔性显示 | 外观、平整性、耐弯折 |
| 车用电子 | 长期可靠性、耐热、环境稳定 |

## 开源记录方式

每个应用可以拆成：

1. 应用场景；
2. 关键指标；
3. 典型测试方向；
4. 风险项；
5. 数据字段；
6. 报告模板。

## 结论

应用文章适合放在 GitHub 的 `docs/applications/` 目录中，作为项目知识库的一部分。
