# 石墨导热膜基材的工程记录方法

## 摘要

石墨导热膜基材涉及厚度、发泡、压延、外观、热扩散与导热性能等多个指标。  
由于具体配方与工艺窗口通常具有商业敏感性，开源项目应只提供通用记录框架，不公开内部参数。

## 建议记录维度

| 维度 | 示例 |
|---|---|
| 产品 | 通用产品代号 |
| 厚度 | 合成或区间化厚度 |
| 发泡状态 | 通用描述 |
| 压延结果 | 通用描述 |
| 外观观察 | 通用描述 |
| 性能指标 | 可使用脱敏或区间化数据 |
| 备注 | 非机密说明 |

## 可以公开的内容

- 记录表格式；
- 数据清洗方法；
- 趋势图结构；
- 统计方法；
- 报告模板。

## 不建议公开的内容

- 真实客户要求；
- 真实批次与卷号；
- 真实配方；
- 真实工艺窗口；
- 商业价格；
- 内部缺陷追溯。

## 结论

石墨导热膜相关知识适合以“工程记录框架”的方式开源，而不是公开具体商业工艺。
