# PI 膜相关应用场景地图

## 目的

本文件用于把 PI 膜、TPI/MPI、石墨导热膜基材、电子级薄膜等方向的应用场景整理成公开、通用、非机密的知识地图。

## 应用地图

| 方向 | 典型下游 | 关注指标 | 适合开源的内容 |
|---|---|---|---|
| 电子级 PI 膜 | FPC / FCCL | 尺寸稳定、力学、外观 | 指标字段、报告模板 |
| 高尺寸安定膜 | 细线路 / 封装 | CTE、收缩、平整性 | 应用拆解、数据结构 |
| 石墨导热膜基材 | 散热材料 | 厚度、发泡、压延、导热 | 记录框架、统计模板 |
| TPI / MPI | 高频高速 | Dk/Df、吸湿、可靠性 | 应用场景说明 |
| 保护膜 / 复合膜 | 封装、显示 | 耐热、剥离、残胶 | 测试记录模板 |

## 注意

本文件仅作为开源应用地图，不代表任何公司商业规划或客户开发计划。
