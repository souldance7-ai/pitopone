# 项目定位：PI 膜产业化工程工具包

## 一句话说明

本项目不是公开公司工艺，也不是发布商业机密，而是把 PI 膜产业化与制造业工程中常见的通用工作流，整理成开源化、模板化、可复用的工具包。

## 为什么要做这个项目

PI 膜、聚合物薄膜、石墨导热膜基材、电子级基材膜、TPI/MPI 复合膜等领域高度专业。  
这些领域的工程资料通常不是以软件项目的形式存在，而是分散在：

- 生产日报；
- 周报；
- 试验报告；
- Excel 表格；
- 质量异常记录；
- 设备商资料；
- 客户验证反馈；
- 工程师个人经验。

这导致一个问题：

> 很多知识可以被重复使用，但缺少统一结构、统一字段、统一报告模板与开源工具。

因此，本项目重点做三件事：

1. 把生产和质量数据变成统一字段；
2. 把统计逻辑变成可复用脚本；
3. 把工程判断变成可维护文档与报告模板。

## 适合的开源表达方式

适合公开：

- 字段结构；
- 合成数据；
- 数据统计方法；
- 报告模板；
- 通用质量分析方法；
- 制程验证记录框架；
- 开源脚本。

不适合公开：

- 真实配方；
- 真实工艺窗口；
- 客户资料；
- 公司产能与成本；
- 内部异常追溯；
- 商业谈判信息。

## 项目价值

这个项目服务的是小众但真实存在的工程场景。  
它的价值不一定体现在 GitHub 星星数，而是体现在帮助工程人员把零散资料转成稳定、透明、可追溯的工程输出。
