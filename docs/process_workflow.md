# General PI Film Industrialization Workflow

This document describes a generalized workflow for organizing production and quality data in polyimide film industrialization.

## Scope

The workflow is intended for open-source engineering data organization and does not include proprietary recipes or confidential manufacturing parameters.

## Typical data sources

- daily production records;
- roll-level output records;
- quality grade data;
- process trial summaries;
- engineering observation notes;
- weekly technical reports.

## Recommended workflow

1. Collect production records.
2. Normalize product names.
3. Normalize grade names.
4. Summarize daily output by product.
5. Calculate grade weight ratio.
6. Identify abnormal grade distribution.
7. Convert results into a technical report.
8. Record suggested engineering actions.

## Output examples

- production summary CSV;
- product grade statistics table;
- A-grade yield trend;
- weekly engineering report;
- process trial report.
