# Contributing

Contributions are welcome.

This project focuses on reusable, non-confidential engineering workflows for manufacturing data organization and technical reporting.

## Please do not submit

- proprietary process recipes;
- company-specific production parameters;
- customer data;
- private equipment settings;
- personal information;
- confidential manufacturing records.

## Good contributions include

- anonymized or synthetic example datasets;
- general-purpose engineering templates;
- data cleaning scripts;
- report generation utilities;
- documentation improvements;
- multilingual usage examples;
- quality analysis methods that do not disclose proprietary details.

## Contribution style

Please keep examples practical, clear, and reusable for engineers who may not be full-time software developers.
